# Task 2: Credit Card Fraud Detection
This project focuses on detecting fraudulent credit card transactions using a Random Forest machine learning model.
## Objective
To identify whether a transaction is fraudulent or not based on transaction data.
## Steps Involved
- Data preprocessing and cleaning
- Handling imbalanced data using undersampling
- Feature engineering (age extraction from DOB)
- Model training using Random Forest
- Model evaluation using confusion matrix and classification report
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
## Result
The model successfully detects fraudulent transactions with good accuracy and balanced performance.
