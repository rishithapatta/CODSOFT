# Customer Churn Prediction
This project is a Machine Learning model that predicts whether a bank customer is likely to leave the bank or continue using the services.

## Project Objective
The main objective of this project is to analyze customer data and predict customer churn using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Random Forest Classifier

## Features Used
- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Credit Card Status
- Active Member Status
- Estimated Salary

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Encoding Categorical Data
4. Train-Test Split
5. Model Training
6. Prediction
7. Accuracy Evaluation
8. Confusion Matrix Visualization

## Model Accuracy
Achieved approximately 86% accuracy.

## Output
- Customer churn prediction
- Confusion matrix
- Feature importance visualization
