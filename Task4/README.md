# Spam SMS Detection
This project is a Natural Language Processing (NLP) and Machine Learning based application that classifies SMS messages as spam or legitimate (ham).

## Project Objective
The objective of this project is to build an AI model that can automatically detect spam SMS messages using text classification techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## NLP Technique Used
- TF-IDF Vectorization

## Machine Learning Algorithm
- Multinomial Naive Bayes

## Project Workflow
1. Data Loading
2. Data Preprocessing
3. Label Encoding
4. TF-IDF Feature Extraction
5. Train-Test Split
6. Model Training
7. Prediction
8. Accuracy Evaluation
9. Confusion Matrix Visualization

## Model Accuracy
Achieved approximately 97% accuracy.

## Output
- Spam message classification
- Classification report
- Confusion matrix visualization
